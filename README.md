# 个人记账 Web 应用

一个基于原生 JavaScript + LocalStorage 实现的个人记账应用，适合前端新手学习和使用。无需后端服务器，所有数据均保存在浏览器本地。

## 项目特点

- **零框架依赖**：仅使用 HTML、CSS、JavaScript 原生技术实现
- **本地数据存储**：使用 LocalStorage 持久化账单、分类和设置数据
- **响应式设计**：同时适配移动端和桌面端，支持深色/浅色主题切换
- **核心功能完整**：支持账单记录、分类管理、数据统计、搜索筛选等功能
- **数据安全**：支持 JSON 格式数据导入导出，方便备份和恢复

## 运行方式

### 方式一：直接打开（最简单）

1. 使用文件管理器找到 `bookkeeping-app/index.html`
2. 双击文件，使用浏览器打开即可使用

> 注意：由于浏览器安全策略，直接通过 `file://` 协议打开时，LocalStorage 通常可以正常工作。如遇问题，请使用方式二。

### 方式二：使用本地服务器（推荐）

#### 使用 VS Code Live Server 插件

1. 在 VS Code 中安装 **Live Server** 插件
2. 右键点击 `index.html`，选择 **"Open with Live Server"**
3. 浏览器会自动打开应用

#### 使用 Node.js 的 http-server

1. 确保已安装 Node.js
2. 在项目目录下执行以下命令：

```bash
npx http-server -p 8080
```

3. 打开浏览器访问 `http://localhost:8080`

#### 使用 Python 临时服务器

如果已安装 Python，可以在项目目录下执行：

```bash
# Python 3
python -m http.server 8080

# Python 2
python -m SimpleHTTPServer 8080
```

然后访问 `http://localhost:8080`

## 功能说明

### 1. 首页概览

- 显示当前月份的总收入、总支出和结余
- 展示支出分类占比环形图
- 显示最近 5 条账单记录
- 支持切换月份查看历史数据

### 2. 账单记录

- 点击右下角「+」按钮添加新账单
- 支持选择收入/支出类型
- 支持选择分类、填写金额、日期和备注
- 在账单列表页可编辑或删除已有账单
- 支持按关键词搜索、时间范围、收支类型、分类筛选

### 3. 分类管理

- 默认已创建常用支出和收入分类
- 支持添加自定义分类（可设置名称、图标、颜色）
- 支持编辑和删除自定义分类
- 默认分类不可删除，避免数据异常
- 删除分类时，相关账单会自动移至「其他」分类

### 4. 数据统计

- 支持按本月、本年、全部三个维度统计
- 显示总收入和总支出
- 展示支出分类 TOP 5 排行
- 展示支出分类占比图表

### 5. 设置

- 浅色/深色主题切换
- 导出数据为 JSON 文件备份
- 从 JSON 文件导入数据恢复
- 清空所有数据（谨慎操作）

## 数据存储说明

应用使用浏览器的 LocalStorage 存储数据，主要包含以下 Key：

| Key | 说明 |
|-----|------|
| `bookkeeping_bills` | 账单记录数组 |
| `bookkeeping_categories` | 分类信息数组 |
| `bookkeeping_accounts` | 账户信息数组（预留） |
| `bookkeeping_settings` | 应用设置对象 |

### 数据备份建议

由于 LocalStorage 数据保存在当前浏览器中，建议定期通过「设置」页面的「导出数据」功能备份 JSON 文件。在以下情况下数据可能会丢失：

- 清除浏览器缓存和数据
- 使用隐私/无痕模式浏览
- 卸载或重装浏览器
- 更换电脑或浏览器

## 学习建议

本项目代码结构清晰，适合前端新手阅读和学习：

1. 先查看 `index.html` 了解页面结构
2. 再查看 `css/style.css` 学习响应式布局和主题切换
3. 最后重点阅读 `js/app.js`，理解数据存储、视图渲染和事件处理逻辑
